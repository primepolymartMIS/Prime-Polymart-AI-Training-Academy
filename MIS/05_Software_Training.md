# 05 - Software Training

## Purpose

The purpose of this document is to provide complete software training for MIS Executives at Prime Polymart Pvt. Ltd.

This module introduces all business-critical software used in daily operations and explains their purpose, business importance, primary functions, and expected competency level.

The objective is to ensure every MIS Executive can confidently use the required software, understand how each application supports business processes, and identify where each software fits within the company's overall workflow.

This document serves as the foundation for software-related learning. Detailed operating procedures, process-specific instructions, and troubleshooting guides are covered in their respective SOPs.

## Learning Objectives

After completing this module, the trainee should be able to:

- Identify all software used by the MIS Department.
- Understand the purpose of each software application.
- Explain how each software supports business operations.
- Navigate business-critical applications confidently.
- Recognize the relationship between software, business processes, and SOPs.
- Follow company standards while using each application.
- Identify common software-related issues and know when to escalate them.
- Understand the expected proficiency level for each software.

## Software Overview

The MIS Department at Prime Polymart uses multiple software applications to manage business operations, automate workflows, analyze data, and support decision-making.

Each software has a specific role within the organization. Every MIS Executive is expected to understand the purpose of these applications, know how they interact with each other, and use them according to company policies and SOPs.

The following table provides an overview of the primary software used in daily operations.

| Software | Primary Purpose | Used By | Importance |
|----------|-----------------|---------|------------|
| Gmail | Business communication | All Employees | High |
| Google Drive | File storage and collaboration | All Departments | High |
| Google Sheets | Data management and reporting | MIS, Sales, Accounts | Critical |
| Google Forms | Data collection | All Departments | High |
| Google Apps Script | Workflow automation | MIS | Critical |
| Microsoft Excel | External reports and data analysis | MIS | High |
| AppSheet | Mobile business applications | All Departments | High |
| Looker Studio | Dashboards and business intelligence | Management, MIS | High |
| DoubleTick WhatsApp | Customer communication and CRM | Sales, CRM | High |
| Runo | Call tracking and communication analytics | Sales, MIS | Medium |

### Expected Competency

Every MIS Executive should:

- Understand the business purpose of each software.
- Navigate each application confidently.
- Follow company standards while using the software.
- Identify common issues and perform basic troubleshooting.
- Know when to escalate software-related problems.
