# 1. Purpose

This Standard Operating Procedure (SOP) explains how to configure, assign, monitor, and review recurring business tasks that occur on a Weekly, Monthly, Quarterly, Half-Yearly, or Yearly basis.

The Periodic Task Management System is maintained separately from the Daily Checklist System to improve performance, reduce spreadsheet size, and simplify recurring task management.

---

# 2. Scope

This SOP applies to all recurring operational, management, and compliance activities that are not performed daily.

Examples include:

- Weekly Reports
- Monthly Reports
- Quarterly Reviews
- Half-Yearly Reviews
- Annual Activities
- Compliance Activities
- Management Reviews
- Stock Audits

---

# 3. Supported Task Frequencies

| Frequency | Code | Example |
|-----------|------|------------------------------|
| Weekly | W | Weekly Sales Summary |
| Monthly | M | Monthly Sales Report |
| Quarterly | Q | Quarterly Business Review |
| Half-Yearly | H | Half-Year Performance Review |
| Yearly | Y | Annual Stock Audit |

---

# 4. Accessing the File

Open the system using any of the following:

- Google Drive
- Google Site
- AppSheet

**File Name**

> **Periodic Task Management System**

---

# 5. One-Time Setup

The initial setup process is identical to **SOP-001 – Daily Checklist System**.

Before using this system, verify the following:

- Script Authorization completed
- Week List updated
- Working Day Calendar updated
- Holiday List updated
- Employee/Doer List updated

If any setup item is incomplete, complete SOP-001 before proceeding.

---

# 6. Task Creation Procedure

## Step 1 – Open Task List

Navigate to the **Task List** sheet.

---

## Step 2 – Enter Task Details

Fill in all required fields:

- Task Name
- Doer Name
- Department
- Frequency
- Planned Date
- Remarks (Optional)

---

## Step 3 – Select Task Frequency

Choose the appropriate frequency.

| Task | Frequency |
|------|-----------|
| Weekly Sales Report | W |
| Monthly MIS Report | M |
| Quarterly Review Meeting | Q |
| Half-Year Performance Review | H |
| Annual Inventory Audit | Y |

---

## Step 4 – Submit Task

Click the **Submit** button.

After submission, verify that the task status changes to:

**Status = Sent**

---

# 7. Verification

Open the **Master** sheet and confirm the following:

- Task created successfully
- Correct employee assigned
- Correct department selected
- Correct frequency assigned
- Correct planned date generated
- No formula or script errors

---

# 8. Dashboard Review

Open the **Dashboard** sheet.

Verify the following metrics:

- Pending Tasks
- Completed Tasks
- Completion Percentage
- Department Performance
- Employee Performance
- Overdue Tasks

Ensure dashboard values match the Master sheet.

---

# 9. Common Errors

| Problem | Possible Cause | Recommended Solution |
|----------|----------------|----------------------|
| Wrong Frequency Selected | Incorrect frequency code | Update the frequency |
| Duplicate Task | Submit button clicked twice | Delete duplicate record |
| Task Missing | Script execution failed | Re-submit the task |
| Dashboard Incorrect | Formula deleted or modified | Restore dashboard formulas |
| Incorrect Due Date | Calendar configuration issue | Update Working Day Calendar |

---

# 10. Troubleshooting

If a recurring task is not generated:

1. Verify Script Authorization.
2. Check the Week List.
3. Verify the Working Day Calendar.
4. Review the Holiday List.
5. Confirm the selected Frequency Code.
6. Re-submit the task.
7. Contact the Senior MIS Executive if the issue remains unresolved.

---
